This directory contains various demos:

*************
** minimal **
*************

This is the minimal demo: see how easy it is to create an EditableGrid.
Only 3 lines of Javascript code, some minimal inline CSS, and here you go!
In this demo, the grid structure and data are obtained from an XML file. 
Of course, you don't have to write this file by hand...
Features: various built-in data types with built-in validation, edition, sorting

************
** attach **
************

This is the exact same demo, but this time the grid structure and data are declared in the HTML file.
Extra column information (mainly the data type) are given using Javascript.

**********
** full **
**********

Features: customer cell renderers and validators, enumeration providers, filtering, paginator, bar and pie charts, date picker, autocomplete
Not featuring: custom cell editors
